# Portfolio Site - Eduardo Sousa

Site de portfólio profissional focado em desenvolvimento web, inteligência artificial e produtos digitais.

## 🚀 Tecnologias

- **Next.js 14** - Framework React com App Router
- **TypeScript** - Tipagem estática
- **Tailwind CSS** - Estilização utilitária
- **Framer Motion** - Animações suaves
- **Lucide React** - Ícones

## 📦 Instalação

```bash
# Instalar dependências
npm install

# Executar em desenvolvimento
npm run dev

# Build para produção
npm run build

# Executar produção
npm start
```

## 🎨 Design

O design é inspirado no site da Porsche, com:
- Tipografia elegante e legível (Inter)
- Paleta minimalista (preto, branco, cinza)
- Espaçamento generoso
- Animações suaves e discretas

## 📁 Estrutura

```
├── app/              # Rotas e páginas (Next.js App Router)
├── components/       # Componentes reutilizáveis
├── data/            # Dados dos projetos
└── public/          # Assets estáticos
```
